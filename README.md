# 📊 Finance Analysis Dashboard - Power BI

## 📌 Overview

This is my **first Power BI dashboard**, created to analyze financial transaction data and transform it into meaningful business insights through interactive visualizations.

The dashboard provides an overview of key financial metrics, transaction trends, customer segmentation, and detailed transaction analysis using Power BI and DAX.

---

## 🚀 Dashboard Features

- Interactive KPI Cards
- Dynamic Year Selection
- Dynamic Slicers & Filters
- Bookmarks & Page Navigation
- Drill-through Transaction Analysis
- Monthly Transaction Trends
- Customer Segment Analysis
- Transaction Status Analysis
- State-wise Financial Analysis
- Interactive Tables & Charts

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling

---

## 📈 DAX Concepts Used

- SUM()
- COUNT()
- CALCULATE()
- DIVIDE()
- SELECTEDVALUE()
- SWITCH()
- Variables (VAR)
- Dynamic Measures
- Percentage Calculations
